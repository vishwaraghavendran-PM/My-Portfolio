   - Clinical documentation

3. Feature Engineering  
4. Model Training  
5. Model Registry  
6. Real-Time Scoring Engine

10. Feedback Loop

Vitals/Labs → Ingestion → Feature Engineering → ML Scoring → Alert Engine → Clinician → Feedback → Retraining.
